#TinyBOYA-spreadsheet-gas-googleform

[Features]
This is a household account book by Spreadsheet and GAS and Google Form.
This is Japanese base.

SpreadSheet Function
 Summary sheet
   This sheet is the monthly summary reports.
   You can see an overview for each subject.
 Monthly Sheets
   You can see and input and edit the account data.
   
 
Google Form Function
 Entry an account data by smart phone or web browser

[概要]
Googleスプレットシート、Googleフォーム、Google Apps Scriptを利用した家計簿アプリです。
以下の理由からGoogleフォームから家計簿を入力できるようにしました。
・入力はスマホがメイン
・スマホでスプレットシートを表示すると見にくい
・スマホでGASを動かせない

24日絞め、25日スタートで各月シートを定義しております。
例えば、Januaryシートの開始と終了は以下の通りです。
　開始日：2018/12/25
  締め日：2019/1/24
